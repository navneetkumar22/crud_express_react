# **CRUD APPLICATION**
This is a simple Full-Stack CRUD application which provides a user to Create, Read, Update and Delete data functionality.

This is designed using Express JS as backend and React is used as frontend.

<br>

## Tech Stack Used

### Frontend
* React Js

### Backend
* Node Js (Express)

### DataBase
* MongoDB (Mongoose)
